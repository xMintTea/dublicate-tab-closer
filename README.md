# Dublicate Tab Closer
Простое не скомпилированное, dev-mode расширение для браузера для закрытие повторяющихся вкладок вкладок.

# Принцип работы:
## При запуске
1. Получает список всех активных вкладок в браузере.
2. Записывает каждый уникальный url в список.
3. Если нашлась вкладка с одинаковым url, то эта вкладка добавляется в список повторок
4. Количество повторок отображается в менюшке
## При нажатии кнопки
1. Скрипт пробегает по списку с найденными повторками и закрывает все повторяющиеся вкладки


# Установка
1. Скачайте архив и распакуйте его где-нибудь
2. Зайдите в браузер и введите `browser://extensions/.` в окно поиска.
3. Активируйте "Режим разработчика"
4. Нажмите на "Упакавать решение" и выберите папку, в которое распаковали скачанный архив
5. Нажмите на "упакавать решение" и можете пользоваться

* Каждый раз при открытии браузера вам будет показываться предупреждение об включенном режиме разработчика. Задаолбливает, но жить можно.



# Пример:
Список открытых вкладок:
1. https://www.youtube.com/watch?v=dQw4w9WgXcQ
2. https://www.youtube.com/watch?v=dQw4w9WgXcQ (дубликат 1)
3. https://www.youtube.com/watch?v=L_jWHffIx5E (ссылка на другое видео)

Скрипт отметит 2, как дубликат и при нажатии на кнопку закроет 2 вкладку.

* Скрипт не хранит ваши вкладки, не передаёт никуда информацию о них и т.д. Ему не интересно, что вы там такое смотрите.
